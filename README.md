# Client from the Bahamas

## How to run

### Install dependencies

```
npm install
```

### Setup MongoDB

1. Install MongoDB (if you do not have installed already)

![Guide](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/)

2. Start MongoDB (on macOS)

```
mongod
```

### Run tests

```
npm test
```

### Run program

```
node app.js
```
